
Overview:

This game is a FPS game that is controlled with the mouse and keyboard. WASD to move, Space to jump, left click to fire. The 1 and 2 keys switch between your active weapons. The `/~ key opens the log. You lose when enough enemies reach the other side of the arena from where they spawn.

Model folder contains the data for the main game objects - the bullets, the guns, the player, and the enemies, as well as the basic node data.
