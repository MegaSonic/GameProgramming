
Overview:

This game is a FPS game that is controlled with the mouse and keyboard. WASD to move, Space to jump, left click to fire. The 1 and 2 keys switch between your active weapons. The `/~ key opens the log. Whenever an enemy reaches the other side of the arena, you lose some health. You lose when your health reaches 0.

C key toggles an overhead view camera that loops around the arena.
V key toggles a third person view camera.

Model folder contains the data for the main game objects - the bullets, the guns, the player, and the enemies, as well as the basic node data.

In-game UI has 3 elements: a player health bar, the current amount of time that you've survived in the level, and the number of enemies you've killed.
